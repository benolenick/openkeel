# Archive — old LLMOS GUI prototypes

The canonical live LLMOS GUI is **`docs/llmos_v2.html`** (one directory up). It is
wired to the real backend endpoints (`/api/chat`, `/api/terminal`, `/api/workshop`,
`/api/learning/retest`, `/api/hyphae/status`, `/api/learning/stats`).

Files in this folder are superseded prototypes kept for reference only:

- `llmos_prototype.html` — original layout sketch (2026-04-05).
- `llmos_wireframes.html` — wireframe exploration (2026-04-05).

Do not edit. If you want to change the live UI, edit `docs/llmos_v2.html`.
